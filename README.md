# always-online

## Prerequisites

```sh
Pyhton = 3.x

```

## Setp 1

Install vertual environment

```sh
cd always-online

python3 -m virtualenv venv

```

## Step 2

Activate virtual environment

```sh
source venv/bin/activate

```

## Step 3

Install all the required dependencies

```sh
pip install -r requirements.txt

```

## Step 4

Run the script

```sh
python run.py

```
